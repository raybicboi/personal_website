Sports Analytics Capstone Golf

Developer

Over the years, golf, like other sports, is being increasingly analytic-based. Following a previous group's findings of a regression formula that can calculate "strokes-gained" per golf swing, my project group was tasked to create an application for the CMU varsity golf team that integrates the formula in a tournament setting.

Created using R and Leaflet, this interface is able to show golf courses using map coordinates, place pin location and swings on click, save and load golf swing data using a nested file system, and return a report of all marked locations its "strokes-gained" score based on the afore-mentioned formula.
